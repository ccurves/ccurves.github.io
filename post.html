Parabathai ChatApp

Few months ago I lost my phone and was looking for a way to still communicate with my friends and I couldn’t use any social media platform due to a restriction my school placed on the Wi-Fi. So I made a simple chat app using Node for the backend and React for the frontend.

I made use of socek.io for real time communication, a user can create a username and a room thereby becoming an admin and other users can join that room. And what is a chatapp without emojis? so i added emojis using React-Emojis

I made use of material-ui for my frontend framework. The frontend of the app was delpoyed using Netlify and the backend was delpoyed using Heroku.
